# README

This is an editor / demo for [coons-patch](https://github.com/undistraction/coons-patch).

## Install

```

pnpm install

```

## Run in development

```

pnpm run dev

```

## Build package

```

pnpm run build

```

## Preview build

```

pnpm run preview

```

## Run tests

Tests are written using Jest

```

pnpm run test

```

## Lint

```
pnpm run lint-prettier
pnpm run lint-eslint
```
